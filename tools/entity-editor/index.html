Welcome to Sajou Theme Editor

Build complete visual themes in 3 tabs:

  1. Assets Drop your sprite folder to import PNG/SVG files. Browse and search your asset library.
  2. Entities Create visual entities (units, buildings, effects). Configure static sprites or animated spritesheets with live preview.
  3. Scene Build your scene layout visually: set the ground, drag decorations, place named positions, and draw routes between them.

Use Export .zip to download everything, or Import JSON to continue editing.
