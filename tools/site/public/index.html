The 3-layer architecture. The choreography layer is the product.

What sajou does

Declarative choreographies

JSON step sequences that describe what happens visually when a signal arrives. Move, spawn, fly, flash, draw beams — composed by humans or by AIs.

MCP Server — 20 tools

Any MCP-compatible AI agent can read scenes, compose choreographies, place entities, write GLSL shaders, create p5.js sketches, and control uniforms in real-time.

Visual scene editor

Node-based workspace with pipeline layout, interlocking block choreographer, shader editor, wiring patch bay, and Three.js live run mode.

GLSL shader system

Full shader editor with CodeMirror, uniform annotations (@ui, @bind, @object), multi-pass ping-pong feedback, auto-detect, and choreo→shader wiring.

NEW

p5.js creative coding

Instance mode p5.js sketches with param annotations, choreographer wiring, and MCP control. Three presets included — Particles, Wave, Grid.

NEW

Lighting & particles

Three.js AmbientLight, DirectionalLight, and PointLight array with flicker. CPU-simulated particle emitters with color-over-life, glow, and radial/directional modes.

NEW

Speech bubbles

Canvas2D overlay with streaming typewriter effect for text_delta/thinking signals. Per-entity config for colors, opacity, tail position, retention.

NEW

Binding transitions

Animated property bindings with configurable duration, easing (linear, easeIn, easeOut, easeInOut, arc), and auto-revert. Float properties only.

Open signal protocol

14 well-known types with typed payloads, plus any custom type. WebSocket, SSE, OpenAI, Anthropic, OpenClaw, Ollama, LM Studio, MIDI — all auto-detected.

NEW

Desktop app (Tauri)

Native desktop app for macOS, Windows, and Linux. No browser mixed-content limits — connect directly to localhost AI services.

Wiring & filters

TouchDesigner-style patch bay connecting signals to choreographies to renderers. Wire-level filters: throttle, sample, delta, when, map.

MIDI input

Hardware controllers mapped to shader uniforms and scene parameters. Knobs, faders, and pads for real-time visual control.

AI agent integration via MCP

Any MCP-compatible AI agent can read, compose, and animate sajou scenes. The MCP server exposes 20 tools — an agent connects via stdio, inspects the scene, builds choreographies, and watches them animate in real-time.
