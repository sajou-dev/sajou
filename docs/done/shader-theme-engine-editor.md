# Shader Theme Engine — éditeur GLSL
Tiers: interface
---
Mode GLSL du shader editor implémenté dans le scene-builder.

## Livré
- Éditeur CodeMirror 6 avec coloration GLSL
- Preview temps réel sur quad plein écran (ShaderToy pattern)
- Uniforms exposés comme sliders, color pickers, toggles, xy pads
- Multi-pass ping-pong feedback
- 3 presets intégrés (Minimal Gradient, Reaction-Diffusion, Noise Field)
- Export/import JSON standalone + intégration ZIP scène
- Auto-detect des littéraux GLSL avec Expose/Unexpose toggle

Mergé dans v0.2.0.
