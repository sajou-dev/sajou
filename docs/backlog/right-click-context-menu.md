# Intercepter le clic droit — menu contextuel
Tiers: interface
---
Capturer les événements clic droit (contextmenu) sur le canvas du scene-builder. À terme, afficher un menu contextuel avec des actions adaptées à la cible : propriétés de l'entité, dupliquer, supprimer, changer de layer, etc. Première étape : intercepter l'événement et empêcher le menu natif du navigateur.
