# Boucle d'interaction bidirectionnelle
Tiers: core
---
Clic sur entité → Stage émet signal → Choreographer joue feedback →
signal remonte sur le bus → agent répond → Choreographer joue réponse → Stage anime.
La boucle complète user ↔ agent à travers la scène.
