# Combinateurs de signaux (merge, mix, split)
Tiers: core
---
Permettre de combiner plusieurs flux de signaux entre eux dans le patch bay. Opérations : merge (union de deux flux), mix (interpolation pondérée entre deux valeurs), split (router un signal vers plusieurs destinations selon des critères), zip (combiner les payloads de deux signaux synchrones). Transforme le pipeline mono-flux actuel en graphe multi-flux.
