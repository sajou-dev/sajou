# Panneau de raccourcis clavier
Tiers: interface
---

Ajouter au panneau de settings une section affichant tous les raccourcis clavier utilisés dans le scene-builder.

## Fonctionnalités

- Liste complète des raccourcis actuels (V, E, R, Z, L, K, I, Ctrl+D, etc.)
- Raccourcis **configurables** par l'utilisateur (remapping)
- Persistance des mappings custom (localStorage ou export JSON)
- Recherche/filtre par action ou touche
- Indication visuelle des conflits si deux actions mappées sur la même touche

## Référence actuelle

| Raccourci | Action |
|-----------|--------|
| V | Select tool |
| E | Entity tool |
| R | Route tool |
| Z | Zone tool |
| L | Light tool |
| K | Particle tool |
| I | Toggle isometric/top-down |
| Ctrl+D | Dupliquer |
| Ctrl+C/V/X | Copier/Coller/Couper |
| Del | Supprimer |

(à compléter avec les raccourcis existants dans le code)
