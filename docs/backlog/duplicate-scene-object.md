# Dupliquer un objet dans une scène
Tiers: interface
---
Permettre de dupliquer un objet (entité placée) dans le scene-builder. Sélectionner un objet puis Ctrl+D (ou menu) pour créer une copie décalée avec les mêmes propriétés (sprite, billboard, layer, etc.). Fonctionne aussi pour les point lights et les positions.
